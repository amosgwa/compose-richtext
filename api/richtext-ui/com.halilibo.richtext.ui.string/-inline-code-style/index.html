InlineCodeStyle

@Immutable
data class InlineCodeStyle(val cornerRadius: TextUnit, val verticalPadding: TextUnit, val horizontalPadding: TextUnit, val topMargin: TextUnit, val bottomMargin: TextUnit, val borderStrokeColor: Color, val borderStrokeWidth: TextUnit, val spanStyle: SpanStyle)

Code style block for inline code. This style represents the highlight behind the inline code.

Parameters

cornerRadius

The corner radius of the highlight.

verticalPadding

The padding above and below the text.

horizontalPadding

The padding to the left and right of the text.

topMargin

Positive margin pushes the box downward.

bottomMargin

Positive margin pushes the box upward.

borderStrokeColor

The color of the border stroke.

borderStrokeWidth

The width of the border stroke.

spanStyle

The SpanStyle to use for the text inside the highlight.
